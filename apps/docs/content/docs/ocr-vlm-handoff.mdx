---
title: OCR and VLM handoff
description: Pass converted PNG pages directly into OCR engines or multimodal SDKs.
---

This package stops at image conversion. OCR stays in third-party tooling.

## Tesseract.js

```ts
import { convert } from "@omsimos/pdf-to-images";
import { createWorker } from "tesseract.js";

const worker = await createWorker("eng");
const [page] = await convert("./scan.pdf", {
  pages: [0],
  dpi: 300,
});

const result = await worker.recognize(page.data);
console.log(result.data.text);
```

## AI SDK `generateText`

```ts
import { convert } from "@omsimos/pdf-to-images";
import { generateText } from "ai";
import { openai } from "@ai-sdk/openai";

const [page] = await convert("./invoice.pdf", {
  pages: [0],
  dpi: 300,
});

const { text } = await generateText({
  model: openai("gpt-4.1"),
  messages: [
    {
      role: "user",
      content: [
        { type: "text", text: "Extract all text from this page." },
        { type: "image", image: page.data },
      ],
    },
  ],
});
```

## Design boundary

Keep responsibilities separated:

- `@omsimos/pdf-to-images`: PDF to PNG
- OCR engine or VLM: text extraction or multimodal understanding
- application layer: orchestration, post-processing, persistence

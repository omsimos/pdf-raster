Use this page if you are working on `@omsimos/pdf-to-images` itself inside the
monorepo. If you only want to install and use the package, start with
[Installation](/docs/installation).

## Local development flow

<Steps>
  <Step>
    Download or refresh the local PDFium cache:

```bash
bun run pdfium:download
```

  </Step>
  <Step>
    Build the native package:

```bash
bun run --filter @omsimos/pdf-to-images build
```

  </Step>
  <Step>
    Run the package tests to validate the local native path:

```bash
bun run --filter @omsimos/pdf-to-images test
```

  </Step>
</Steps>

## If the native setup breaks

<Steps>
  <Step>
    Make sure the package is built for the current platform.
  </Step>
  <Step>
    Refresh the local PDFium cache with `bun run pdfium:download`.
  </Step>
  <Step>
    Rebuild the package with `bun run --filter @omsimos/pdf-to-images build`.
  </Step>
  <Step>
    Run `bun run --filter @omsimos/pdf-to-images test` to validate the native
    path.
  </Step>
</Steps>

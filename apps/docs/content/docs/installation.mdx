## Runtime model

<Callout type="warning" title="Server-side only">
  This package is meant for Node.js and Bun. Do not import it into browser
  bundles.
</Callout>

The library is native and server-side:

- supported runtimes: Node.js and Bun
- rendering backend: PDFium
- primary output: PNG `Buffer`s for each page

Use it in:

- backend jobs
- API routes and route handlers
- worker processes
- document-processing services

## Local development in this monorepo

<Steps>
  <Step>
    Prefetch the current platform's PDFium binary.

```bash
bun run pdfium:download
```

  </Step>
  <Step>
    Build the native package.

```bash
bun run --filter @omsimos/pdf-to-images build
```

  </Step>
  <Step>
    Run the package tests if you want to validate the local native setup.

```bash
bun run --filter @omsimos/pdf-to-images test
```

  </Step>
</Steps>
